Power Calculation (Defining powerInd())

These are the functions used to calculate an individual building's power consumption/production (powerInd()). Total power usage for an item is calculated by power usage of all the whole buildings plus the usage of the remainder building:

( floor(buildings) * powerInd(baseMW, maxClock, sloopMult) ) + powerInd(baseMW, maxClock * (buildings - floor(buildings)), sloopMult)

Where:
"buildings" is the number of buildings needed
" baseMW" is a building's normal/base power usage (at 100%)
"maxClock" is the maximum allowed clocking (clocking is used for the remainder)
"sloopMult" is the Production Amplifier

Presets:
Satisfactory: "baseMW * (sloopMult ** 2) * (clock ** clockPowerExp)"*
Linear: "baseMW * sloopMult * clock"
Ceiling: "baseMW * sloopMult * Math.ceil(clock)"
*clockPowerExp is a global constant


Function used to calculate power consumption (positive).



Uses eval() for powerInd(baseMW, clock, sloopMult) where baseMW >= 0.


Function used to calculate power production (negative).



Uses eval() for powerInd(baseMW, clock, sloopMult) where baseMW < 0.
